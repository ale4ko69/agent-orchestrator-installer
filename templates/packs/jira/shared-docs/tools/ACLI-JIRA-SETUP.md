# ACLI Jira Setup (Optional)

Use this guide if your environment has Atlassian CLI integration.

## Typical Use Cases
- Read issue details quickly
- Move issue status (`To Do` -> `In Progress` -> `Done`)
- Add comments and attach execution evidence

## Safety Notes
- Never hardcode tokens in repository files.
- Store credentials in secure local environment only.
- Keep Jira comments concise and evidence-based.

## Suggested Orchestrator Output to Jira
- What changed
- What was verified
- Remaining risks or follow-ups
